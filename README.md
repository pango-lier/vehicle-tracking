## Development


### Starts a dev server

    yarn
    yarn serve

### Analyzes source code

    yarn lint

Automatically fix problems:

    yarn lint:js --fix
    yarn lint:style --fix

### Produces a production-ready bundle

    yarn build

### Install husky plugin

    yarn husky install
